Security Audit

A security audit is a comprehensive assessment of an organization's cybersecurity posture.It involves reviewing policies, systems, access controls, configurations, and compliance with security standards like ISO 27001, NIST, or HIPAA.

Incident Report

An incident report is a detailed, written account of a cybersecurity incident or breach. It documents what happened, when, how it was detected, who was involved, how it was handled, and the impact.

SQL Query

An SQL query is a structured command used to retrieve or manipulate data in a SQL (Structured Query Language) database.

Vulnerability Assessment

A vulnerability assessment is a systematic process of identifying, classifying, and prioritizing security vulnerabilities in systems, networks, and applications.

TryHackMe Journal

A TryHackMe journal is a personal or professional log where cybersecurity learners document what they’ve learned, practiced, and achieved on the TryHackMe platform—a gamified cybersecurity training site.

Incident Handlers Journal

An incident handler's journal is a logbook kept by cybersecurity professionals (usually SOC analysts or blue teamers) during real-time incident response.
